# deal_check_mcp
DealCheck MCP is a payment-gated Model Context Protocol (MCP) server that helps AI agents and users instantly determine whether a product is a good deal, overpriced, or worth waiting on.

# DealCheck MCP ($2 per call)

AI-powered deal decision MCP server using MPP + Tempo payments.

## Features
- $2 per tool call
- No API keys
- Agent-native payments
- Returns actionable decisions

## Run

npm install  
npm run build  
npm start
